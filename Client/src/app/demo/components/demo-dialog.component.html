Welcome to the Expense Tracker Demo

This application (codenamed Augustus) allows users to upload, categorise and visualise their personal expenses.

This is a sandbox environment unique to you so feel free to play around, re-categorise transactions, add and remove categories and whatever else you feel like!

Categorising Transactions

To categorise transactions, select them in the transaction list and use the categorisation form on the right to add a description and select a category.

Managing Categories

Use the manage categories page to add, edit and delete transaction categories.

Visualising Expenses

The dashboard shows visual breakdowns of expenses per category, allowing users to understand how they spend their money.
